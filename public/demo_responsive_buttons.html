Implementation Details

CSS Classes Used:

  • .action-btn - Base button styles
  • .btn-view - View button (blue gradient)
  • .btn-share - Share button (cyan gradient)
  • .btn-download - Download button (green gradient)
  • .btn-delete - Delete button (red gradient)
  • .btn-icon - Icon styling
  • .btn-text - Text label (hidden on mobile)

Responsive Behavior:

  • Desktop (≥768px): Icons + Text, horizontal layout
  • Mobile (<768px): Icons only, horizontal layout
  • Extra Small (<480px): Large icons, 2-column grid
  • Touch: 44px minimum tap targets, active states
  • Accessibility: Keyboard navigation, ARIA labels

How to Use:

<div class="project-actions">
  <button class="action-btn btn-view" onclick="viewVideo()">
    <i class="fas fa-eye btn-icon"></i>
    <span class="btn-text">View</span>
  </button>
  <button class="action-btn btn-share" onclick="shareVideo()">
    <i class="fas fa-share-alt btn-icon"></i>
    <span class="btn-text">Share</span>
  </button>
  <button class="action-btn btn-download" onclick="downloadVideo()">
    <i class="fas fa-download btn-icon"></i>
    <span class="btn-text">Download</span>
  </button>
  <button class="action-btn btn-delete" onclick="deleteVideo()">
    <i class="fas fa-trash btn-icon"></i>
    <span class="btn-text">Delete</span>
  </button>
</div>
